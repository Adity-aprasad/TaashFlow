# Takht — Multiplayer Card Game Platform

A real-time multiplayer card game platform built with Next.js 16, Supabase, and modern web technologies. Play Takht (trick-taking card game) or use the Generic Score Tracker for any game — no account required.

## 🎮 Features

- **No sign-up**: Just enter a name and play (anonymous Supabase auth)
- **Real-time**: Live updates via Supabase Realtime
- **Multiple games**: Takht + Generic Score Tracker (extensible)
- **Rich UI**: Dark luxury theme with animations, charts, leaderboards
- **Mobile-first**: Responsive design, 48px+ tap targets
- **Extensible**: Add new games by dropping a folder + one registry line

## 🚀 Quick Start

### Prerequisites

- Node.js 20+
- Free [Supabase](https://supabase.com) account
- Free [Vercel](https://vercel.com) account (for deployment)

### 1. Clone & Install

```bash
git clone <repo-url>
cd takht-platform
npm install
```
